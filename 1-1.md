1) git init

- git에 입력을 시작하겠다

2) git remote add origin 레포지토리 주소



3) git add .

- .은 디레토리에 있는 모든 파일을 다 추가 하겠다

4) git commit -m "(원하는 내용)"

- 실행이 되는 것임

- -m 은 커밋에 대한 설명을 적어주겠다는 말 : ex) git commit -m "1st"

5) git push origin master

- 준비했던 파일이 다 올라가는 최종 명령
