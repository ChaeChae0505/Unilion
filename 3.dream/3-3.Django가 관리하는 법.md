## URL 관리 
* Django의 URL관리는 urls.py의 urlpatterns에서 담당
* Path의 구조 path('URL', views 내부의 함수, name = "url의 이름"),
      URL ) 페이지 주소(ex. intrduce/, new/)
      함수 ) url이 불렸을 때 실행할 함수 ( ex. views.home)
      name ) 해당 path를 대표하는 이름 (ex. name = "home")
